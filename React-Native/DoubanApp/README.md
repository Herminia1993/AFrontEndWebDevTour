# DoubanApp

参考 [@vczero](https://github.com/vczero) 老师的 [React-Dou](https://github.com/vczero/React-Dou)，实现了一个更新的精简版豆瓣 APP（iOS）。

## Feature

- 使用了 ES 6 语法
- 搜索图书、电影、音乐
- 查看图书、电影、音乐详情
