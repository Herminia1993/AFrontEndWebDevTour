1. 写入 HTML 输出

JavaScript 能够直接写入 HTML 输出流中，比如执行下面的代码：

<script type="text/javascript"> // 直接写入 HTML 输出 document.write("<h3>This is a heading</h3>"); document.write("<p>This is a paragraph.</p>"); </script>

将得到如下效果：

您只能在 HTML 输出流中使用 document.write。 如果您在文档已加载后使用它（比如在函数中），会覆盖整个文档。
