#### browsersync

能让浏览器实时、快速响应您的文件更改（html、js、css、sass、less等）并自动刷新页面。

使用方法：进入index.html 目录下，执行 `browser-sync start --server --browser "Google Chrome" --files "stylesheets/*.css, *.html"` 命令。
